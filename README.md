# 红楼梦新续

> **作者：** 石头记续书人  
> **缘起：** 高鹗续书虽存，然多违曹雪芹原意。今据脂砚斋批语、前八十回伏笔，重续后三十回，以还原著之魂。

## 编撰原则

1. **忠于原意**：依据脂批、判词、曲子伏笔
2. **尊重人物**：不扭曲宝黛钗等核心人物性格
3. **悲剧结局**：贾府彻底败落，"白茫茫大地真干净"
4. **文风统一**：尽力模仿曹雪芹笔法

## 主要情节走向（据脂批推测）

- 第八十一回起：贾府内外交困，经济日窘
- 秋天：贾府获罪，宝玉被羁狱神庙
- 次年春尽：黛玉泪尽夭亡
- 宝玉归来后与宝钗成婚（无"调包计"）
- 元春暴毙、探春远嫁、迎春被虐致死
- 贾府抄没，众人入狱
- 宝玉最终悬崖撒手，出家为僧
- 末回：警幻情榜
